# Text_Corrector
英文和中文文本纠错
### 1. check_en.py为对英文单词进行纠错。假设编辑距离为1或2，假设已知错误的单词，对单词进行纠错。 
### 2. correction_basic.py为使用bert模型进行中文纠错，在假设已知错误词的位置的情况下就行纠错。
